# Yarnify 🧵

"Looped with Love" — A vibrant, responsive landing page for handmade crochet creations, designed to showcase unique, cozy pieces and connect with customers for custom orders.

## 🚀 Project Overview

**Yarnify** is a dedicated web platform built to highlight the art of handmade crochet. The site serves as both a digital gallery and a communication bridge, allowing visitors to explore collections, learn about the brand, and easily inquire about custom, one-of-a-kind crochet pieces.

## ✨ Features

*   **Cozy, Handmade Aesthetic:** A warm and inviting design language that reflects the "Looped with Love" ethos, creating an emotional connection with the user immediately.
*   **Intuitive Navigation:** A clean header and organized layout that guides users through collections, new releases, and customization options seamlessly.
*   **Customer-Focused CTAs:** Strategically placed calls-to-action (like the "DM for custom order" footer) that streamline the ordering process and drive engagement.
*   **Social Integration:** Direct link to the Instagram community (`yarnifyy.in`), bridging the gap between your website and your active social media presence.
*   **Responsive Design:** Optimized for mobile and desktop, ensuring that potential customers can view your creations clearly, whether they are on the go or browsing from home.

## 🛠️ Tech Stack

*   **Structure:** HTML5 (Semantic and clean markup)
*   **Styling:** CSS3 (Modern, responsive layout techniques)
*   **Deployment:** Netlify

## 🎨 Design Philosophy

Yarnify was built to translate the tactile, cozy nature of crochet into a digital experience. The interface prioritizes large, engaging visuals and clear typography, ensuring the handmade quality of your products remains the primary focus while keeping navigation simple and functional.
